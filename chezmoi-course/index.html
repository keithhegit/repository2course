稳定运行和风险边界同样是产品能力的一部分。

代码与规则片段

CODE

The danger is applying source-state changes to the wrong destination context
Security comes from controlling templates, secrets, and host-specific rendering paths
A dotfile manager is really a state transition engine with filesystem side effects
Pulling and applying should be separated mentally even when one command can do both

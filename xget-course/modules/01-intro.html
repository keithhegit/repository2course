产品与价值

Xget 不是简单的反向代理，而是按 Git、Docker、AI、标准 HTTP 等协议分叉处理的边缘加速引擎。

代码与规则片段

CODE

One engine accelerates Git hosting, model hubs, package registries, container registries, and AI inference providers
Architecture splits requests into GitHandler, DockerHandler, AIHandler, and Standard Handler
Protocol-aware behavior covers HTTP/3, range requests, Git traffic, registry flows, and AI API compatibility
Security headers, request validation, and timeout protection are built into the request path
